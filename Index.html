Aprendendo Github

Este repositório tem objetivo deixar registrado meu avanço no estudo dos comandos do github.


Comandos aprendidos

  • git init
  • git add .
  • git commit -m "mensagem"
  • git status
  • git log
  • git remote add origin "link do repositório"
  • git push -u origin main

O que cada comando faz?

Comando Função
git init Inicia um repositório local
git clone [url] Clona um repositório Git existente para o diretório local.
git add . Adiciona os arquivos ao repositório local
git commit -m "mensagem" Salva as alterações no repositório local com uma mensagem descritiva
git status Mostra o status dos arquivos no repositório local
git log Exibe o histórico de commits do repositório local
git remote add origin "link do repositório" Conecta o repositório local a um repositório remoto no GitHub
git push -u origin main Envia as alterações do repositório local para o repositório remoto no GitHub
